3. **https://metamask-airdrop.xyz 여기 들어가도 돼?**

4. **친구가 보내준 링크인데 믿어도 될까? upbit-event.com**

5. **이 암호화폐 거래소 사이트가 공식 사이트인지 검증해줘**

## 복합 사용 예시

1. **이 사이트 검증하고, 위험하면 비슷한 사기 유형도 알려줘**
   - validate_domain + get_trending_scams 조합

2. **binance-kr.com 검증하고, 가짜 거래소 구별법도 알려줘**
   - validate_domain + educate_user(fake-exchange) 조합

3. **요즘 유행하는 사기 알려주고, 예방법도 설명해줘**
   - get_trending_scams + educate_user 조합

4. **이 사이트 검증해보고, 정보가 없으면 신고해줘**
   - validate_domain 후 결과에 따라 report_scam 조합

5. **fake-exchange.com 검증하고, 사기면 바로 신고해줘**
   - validate_domain + report_scam 연계

## 주의사항

- 도메인 검증 결과가 "안전"이라도 100% 보장은 아닙니다
- 항상 공식 URL을 북마크해두고 사용하세요
- 시드 문구(복구 구문)는 절대 입력하지 마세요
- 의심스러운 링크는 먼저 검증 후 접속하세요
